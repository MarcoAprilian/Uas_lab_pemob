## Deskripsi Aplikasi

Aplikasi sederhana yang berfungsi untuk simpan *watchlist* anime seperti MAL (my anime list), adapun fiturnya sebagai berikut:
1. login/signup
2. search anime
3. favorit anime
4. filter anime
5. ganti password
